
### 运行环境:

  node: v18.19.0  
  yarn: v1.22.19

### 开发姿势
  1. 安装git
  2. git clone https://github.com/johnsonwong666/profile-demo.git
  3. 进入项目目录,执行yarn 
  4. 执行yarn start启动项目
